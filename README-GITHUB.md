# VB-Swap Mod for Minecraft 1.21.11

🎮 **Swap your mainhand and offhand items instantly using a chat command - even while in GUIs!**

## ✨ Features

- Type `.vb-swap` in chat to swap held item with offhand
- **Works in ANY GUI** - chests, crafting tables, furnaces, etc.
- Client-side only - works on vanilla servers
- No keybind needed - just type the command!
- Built for Minecraft **1.21.11** (Mounts of Mayhem)

## 📥 Download

**Option 1: Pre-built JAR (Easiest)**
1. Go to the [Actions](../../actions) tab
2. Click on the latest successful build (green checkmark)
3. Download the `vbswap-mod` artifact
4. Extract the ZIP to get `vbswap-1.0.0.jar`

**Option 2: Build it yourself**
1. Make sure you have Java 21 installed
2. Clone this repository
3. Run `./gradlew build` (Linux/Mac) or `gradlew.bat build` (Windows)
4. Find the JAR in `build/libs/`

## 🔧 Installation

1. Install [Fabric Loader](https://fabricmc.net/use/) for Minecraft 1.21.11
2. Install [Fabric API](https://www.curseforge.com/minecraft/mc-mods/fabric-api)
3. Download the mod JAR (see above)
4. Place `vbswap-1.0.0.jar` in your `.minecraft/mods/` folder
5. Launch Minecraft with Fabric!

## 🎯 Usage

1. Open any inventory or GUI (or just stand normally)
2. Make sure you're holding an item you want to swap
3. Type `.vb-swap` in the chat
4. Press Enter
5. **Boom!** Your mainhand and offhand are swapped! 🔄

## 🛠️ Requirements

- **Minecraft:** 1.21.11
- **Fabric Loader:** 0.18.1+
- **Fabric API:** 0.111.0+1.21.11
- **Java:** 21+
